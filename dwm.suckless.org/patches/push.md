# PUSH UP/DOWN

## Description

This patch provides a way to move clients position inside the clients list.

     #include "push-5.3.c"

     { MODKEY|ControlMask,           XK_j,           pushdown,       NULL }, \
     { MODKEY|ControlMask,           XK_k,           pushup,         NULL }, \
