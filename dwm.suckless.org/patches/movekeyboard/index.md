========

Description
-----------
This Patch adds the ability to move floating windows on the x and y axis with the keyboard, instead of only the mouse. This is achieved by adding the two functions "movekeyboard-x" and "movekeyboard-y".

Default bindings:
modkey + ctrl + l : x + 20
modkey + ctrl + h : x - 20
modkey + ctrl + j : y + 20
modkey + ctrl + k : y - 20
