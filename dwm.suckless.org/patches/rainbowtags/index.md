rainbowtags
====

Description
-----------
Allows each tag to be an independent/unique color
when selected.

Two arrays of colors (tagselfg, tagselbg) are used
for tags in place of selfgcolor and selbgcolor.

Inspired by colorful 90s & 2000s software.
