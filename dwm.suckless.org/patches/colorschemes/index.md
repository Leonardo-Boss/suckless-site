colorschemes
============

Description
-----------
This patches provides the ability to cycle between any number of colorschemes
defined in config.h.

Usage
-----
Append alternative colorschemes to the `colors[]` array.

Default keybindings
--------------------
* `s` - Cycle between colorschemes.
