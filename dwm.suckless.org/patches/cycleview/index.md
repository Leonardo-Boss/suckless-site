Focus Visible Tags Stacks
=========================

Description
-----------
This patch allow to cycle through all the tags of the active monitor

Optional you can set the direction with 0 or 1

Default keybinding
-------------------
* `Mod+Shift+j` 
* `Mod+Shift+k`
