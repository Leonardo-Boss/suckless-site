# ATTACHASIDE

## Description

Make new client get attached and focused in the stacking area instead of
always becoming the new master. 
Basically an attachabove mod.

## Example

	Original behaviour :
	+-----------------+-------+
	|                 |       |
	|                 |   P   |
	|                 |       |
	|        N        +-------|
	|                 |       |
	|                 |       |
	|                 |       |
	+-----------------+-------+


	New Behaviour :
	+-----------------+-------+
	|                 |       | 
	|                 |   N   | 
	|                 |       | 
	|        P        +-------+
	|                 |       |
	|                 |       |
	|                 |       |
	+-----------------+-------+


	+-----------------+-------+
	|                 |       |
	|                 |   P   |
	|                 |       |
	|                 +-------+
	|                 |       | 
	|                 |   N   | 
	|                 |       | 
	+-----------------+-------+
