ARGBColor
==========

->[![Screenshot](dwm-alpha.png)](dwm-alpha.png)<-

Description
-----------

Allow dwm to have translucent bars, while keeping all the text on it opaque,
just like the [alpha-patch for st](http://st.suckless.org/patches/alpha).
